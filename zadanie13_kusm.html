Def body:


Tag body definiuje treść dokumentu. Element body zawiera całą zawartość dokumentu HTML, taką jak tekst, nagłówki, akapity, zdjęcia, grafika, hiperłącza, tabele, listy itp.
Uwaga: dokument HTML może zawierać tylko jeden element body.
